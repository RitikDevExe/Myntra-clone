\# 🛍️ Myntra Clone (HTML \& CSS)



A visually accurate clone of the Myntra e-commerce website built using pure HTML and CSS.  

This project focuses on layout structuring and UI design.



\---



\## 🚀 Live Demo


🔗 https://RitikDevExe.github.io/Myntra-clone


\---



\## ✨ Features



\- Clean and modern UI inspired by Myntra

\- Multi-page navigation:

 - Homepage

 - Product listing page

 - Product detail page

\- Product cards with pricing & discounts

\- Filter sidebar UI

\- Add to Bag & Wishlist buttons (UI only)



\---



\## 🛠️ Tech Stack



\- HTML5

\- CSS3
